## ✨ Features

- 🚀 High-performance async DNS testing
- 📊 Real-time benchmarking with median, mean, min, max, std dev
- 🎯 Multi-server comparison (Google, Cloudflare, Quad9, OpenDNS, etc.)
- 📈 Advanced metrics: Jitter analysis, P95/P99 percentiles
- 💾 JSON export functionality
- ⚙️ Fully customizable settings
- 🎨 Modern dark-mode GUI built with PyQt6
- 🔄 Auto-update checker

- Enable jitter analysis and percentile reporting
- Toggle auto-update checks
- Export results to JSON
